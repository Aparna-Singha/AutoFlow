**AutoFlow** is a no-code automation platform that lets anyone automate complex tasks by writing simple, human-readable instructions.

```yaml
workflow:
  - agent: WebScraper
    task: "Fetch AI news"
  
  - agent: Summarizer
    task: "Summarize in 3 lines"
  
  - agent: Mailer
    task: "user@example.com"
